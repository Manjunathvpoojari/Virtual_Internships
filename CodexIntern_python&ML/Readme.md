## 📌 Slab 2 – Intermediate Project

### 🔹 Task 1 – Voice-Activated Personal Assistant 🎙️

- Developed a **voice-based AI assistant using Python**  

#### 🧠 Core Implementation:
- Used `speech_recognition` library to capture voice input  
- Converted speech to text using Google Speech API  
- Used `pyttsx3` for text-to-speech output  
- Designed a command-processing system to map user input to actions  

#### ⚙️ Functional Modules:
- **Reminder System**
  - Parses time from voice commands  
  - Stores and triggers reminders  

- **Weather & News Integration**
  - Fetches real-time data using APIs  
  - Displays meaningful responses  

- **Utility Features**
  - Time and date retrieval  
  - Basic calculations  
  - Opening websites  
  - System status info  

- **Interactive Features**
  - Joke generator  
  - Continuous listening loop  
  - Error handling for voice recognition failures  

#### 💡 Advanced Aspects:
- Background thread for reminder checking  
- Command parsing using natural language patterns  
- Modular class-based design (`VoiceAssistant`)  

👉 This project helped me understand real-world AI systems combining NLP, APIs, and automation.
